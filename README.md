# Aventurier 投资回报率计算器

专业的 Aventurier 洗地机投资回报率计算器。

## 主要功能
- 计算工资、水费、化学品成本节省
- 4种机型：Youth（1-2电池）和 Pro（1-2电池）版本
- 响应式设计，支持所有设备
- Aventurier 品牌企业色彩

## 使用方法
1. 选择机器型号
2. 输入清洁参数
3. 配置成本参数
4. 点击"计算"

## 技术规格
基于 Aventurier Artist 1 的实际规格：
- 清洁效率：800-1200 平方米/小时
- 工作时间：Eco模式75分钟 / Turbo模式60分钟
- 水箱容量：4升
- 工作宽度：40厘米
- 噪音水平：60分贝（Eco模式）
